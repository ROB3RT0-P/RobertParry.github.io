Background

If one phrase could represent me, it would be "Programming gives me purpose and music fulfills me."
My introduction to the world of computers was through the use of an old Windows PC packed with a bunch retro emulators. Emulation is something I'm still very interested in, but I'm fully aware that it is more of a passion project due to legalities. However, I still consider emulation a great way to learn and a provider of a useful skill sets to have as a programmer. I first started programming because of my interest in video games. I wanted to be a game designer and programmer when I was older. The first game I made was an FPS when I was 9-10 years old using a program called 'Alice'. Later on when I was 15 Years old I started programming using 'Unreal Engine 4'. I used resources and documentation from the UE4 website to create the games I made.

...Nowadays, I'm still interested in games, but my perspective of what is interesting has broadened vastly. I have gained interest in cyber security and software engineering as well as reverse engineering retro consoles. The reason I became interested in cyber security is because of how technology has been progressing in recent years. Technology is a tool, and tools like this can be exploited. Therefore, I think it's very valuable to have a good understanding of cyber security. I think the best way to do this is to learn about how technology works at it's roots. So recently I have been researching Operating Systems, VPN services and data encryption.
